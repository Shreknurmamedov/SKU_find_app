# Android app

Android-клиент будет отвечать за:

- запись видео через CameraX;
- live-контроль качества: смаз, освещение, скорость движения, расстояние до товара;
- подсказки оператору;
- загрузку видео/keyframes на backend;
- отображение отчета и зон пересъемки.

Первый backend-прототип уже находится в `backend/`. Android-каркас лучше создавать после фиксации минимальной версии Android, целевых устройств и режима работы: online-only или partial offline.
